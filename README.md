## Featured Work
### Snowflake Medallion Architecture Pipeline
Built a multi-stage data platform that ingests SQL Server and Azure Blob data into Snowflake, applies layered transformations, and prepares analytics-ready outputs for BI and forecasting use cases.

### Snowflake Badge 3 Streamlit App
Created an interactive Streamlit application backed by Snowflake that lets users assemble smoothie orders, inspect nutrition data, and write orders back to Snowflake.
